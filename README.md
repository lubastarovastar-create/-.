# Клубный Портал Колледжа

## Краткое описание
Веб-сервис для студентов, который помогает найти подходящий кружок или клуб в колледже. На сайте есть каталог всех кружков с описанием, расписанием и контактами. Можно сразу подать заявку на вступление. Всё в одном месте — не нужно искать информацию в разных чатах и соцсетях.

## Цель сервиса
1. Собрать информацию о всех кружках колледжа на одной платформе
2. Упростить процесс записи в кружки (онлайн, без бумажных заявок)
3. Сделать актуальное расписание доступным для всех студентов
4. Увеличить число участников внеучебных активностей

## Инструкция запуска
1. Скачайте папку с проектом
2. Найдите файл `index.html` в папке
3. Откройте его двойным кликом в любом браузере (Chrome, Firefox, Edge)
4. Сайт загрузится и будет готов к работе

## Описание реализованного функционала

### 1. Страницы
✅ **Главная страница (index.html)** — приветствие и ссылки на другие страницы  
✅ **Страница описания (about.html)** — информация о проекте и команде  
✅ **Основная страница (clubs.html)** — каталог всех кружков с фильтрацией  
✅ **Страница с формой (join.html)** — форма записи в кружок  

### 2. Пользовательский интерфейс
✅ **Единый стиль** — одинаковые цвета, шрифты и кнопки на всех страницах  
✅ **Меню навигации** — вверху каждой страницы есть ссылки на все разделы  
✅ **Адаптивный дизайн** — сайт нормально выглядит на телефоне и компьютере  
✅ **Читаемый текст** — крупные буквы, контрастные цвета  

### 3. JavaScript-функции
✅ **Интерактивные элементы** — кнопки, фильтры, форма реагируют на клики  
✅ **Обработка ввода** — поиск по кружкам работает при наборе текста  
✅ **Валидация формы** — проверка email и обязательных полей  
✅ **Динамическое обновление** — фильтрация кружков без перезагрузки страницы  

✅ **Основной модуль: ФИЛЬТРАЦИЯ КРУЖКОВ**
- Поиск по названию (начинает искать при вводе каждой буквы)
- Фильтры по категориям: "Спорт", "Творчество", "Наука", "Другое"
- Кнопка "Показать все" для сброса фильтров

### 4. Работа с данными
✅ **Данные о кружках** — хранятся в файле `data/clubs.json` (15 кружков)  
✅ **Заявки пользователей** — сохраняются в LocalStorage браузера  
✅ **Структура данных кружка**: название, описание, категория, расписание, контакты, место проведения# -.
